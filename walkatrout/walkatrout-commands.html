Walka-Bot list of commands

I'm going to rewrite all Walka-Bot's code, and I may remove some commands (And add new ones aswell).

Walkahelp Shows this list.
Walkamoji Sends some Walkatrout emojis.
Walkanize Give yourself the Walkatrout role.
Walkaserver Get an invite to my Discord server.
Walkainvite Invite Walka-Bot to your server.
Walkahmmm Sends some Rook emojis.
Walkarandom Sends a random Omnitrix emoji.
Walkaclip Sends a random Walkatrout clip.
Walkainfo Sends a short version of the Walkatrout's Wiki.
Walkaverse Sends a random version of Walkatrout (emoji).
Walkaquiz Starts a quiz related to Ben 10.
Walkaomnitrix Use the Original Series Omnitrix.
Walkaquote Sends a Walkatrout's quote from the series.
Walkaaliens Guess the alien on the image.
Unwalkanize Removes your Walkatrout role.

Walka-Bot special commands

I'm going to rewrite all Walka-Bot's code, and remove these commands.

+say message Send a message as Walka-Bot (Permission Manage Messages Required).
+changelog Check the latest updates.
Walkamoji.upload Upload the Walkamoji emoji to your server (Permission Manage Emojis required).
Walkaverse.upload Upload all the Walkaverse emojis to your server (Permission Manage Emojis required, default Walkatrout not included).
Walkaomnitrix.help Extension of the command Walkaomnitrix.

Note: The commands are non case-sensitive.
